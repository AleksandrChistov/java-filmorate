## Описание проекта

Проект Filmorate представляет собой RESTful веб-приложение для управления фильмами и пользователями, реализованное с использованием Spring Boot. Приложение позволяет пользователям добавлять фильмы, ставить им лайки, добавлять друзей и просматривать популярные фильмы.

## Архитектура

Проект реализован по принципам Clean Architecture с разделением на слои:

- **Controller**: Обработка HTTP-запросов
  - `FilmController` - управление фильмами
  - `UserController` - управление пользователями
  - `GenreController` и `MpaController` - управление жанрами и рейтингами

- **Service**: Бизнес-логика приложения
  - `FilmService` - логика работы с фильмами
  - `UserService` - логика работы с пользователями

- **Model**: Модели данных
  - `Film` - модель фильма с полями: id, название, описание, дата релиза, продолжительность, рейтинг (MPA), жанры, лайки
  - `User` - модель пользователя с полями: id, имя, логин, email, дата рождения, друзья

- **DAL (Data Access Layer)**: Доступ к данным
  - Реализованы хранилища для работы с базой данных (DbStorage) и in-memory хранилища
  - Используются RowMapper'ы для маппинга результатов запросов

## Функциональные возможности

### Управление фильмами:
- Добавление нового фильма
- Обновление информации о фильме
- Получение списка всех фильмов
- Получение фильма по ID
- Получение популярных фильмов (по количеству лайков)
- Добавление и удаление лайков

### Управление пользователями:
- Добавление нового пользователя
- Обновление информации о пользователе
- Получение списка всех пользователей
- Получение пользователя по ID
- Добавление и удаление друзей
- Получение списка друзей
- Получение общих друзей с другим пользователем

## Технологии

- Java 21
- Spring Boot 3.2.4
- Spring Web
- Spring Validation
- Lombok
- JUnit 5 (тестирование)

## Запуск проекта

1. Убедитесь, что у вас установлена Java 21
2. Соберите проект с помощью Maven: `mvn clean package`
3. Запустите приложение: `java -jar target/java-filmorate-0.0.1-SNAPSHOT.jar`

После запуска приложение будет доступно по адресу: `http://localhost:8080`
